## Introduction
Rumah Lelang adalah aplikasi berbasis website guna membantu suatu organisasi atau masyarakat dalam memasang/menemukan barang pelelangan. Rumah Lelang dibangun menggunakan Framework PHP Laravel. Enjoy rek !!!

## How to Run
1. run "Composer install"
2. Create a table named "rumahlelang"
3. run "php artisan migrate:fresh"
4. run php artisan ser
5. enjoy!!!
